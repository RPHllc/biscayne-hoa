# Gallery Images

Place gallery images here and reference them in `content/gallery.json`.

Recommended naming:
- yyyy-event-slug-01.jpg
- yyyy-event-slug-02.jpg

Example entry:
{
  "src": "/images/gallery/2024-july-4th-01.jpg",
  "alt": "Fireworks over the bay",
  "caption": "July 4th celebration"
}
